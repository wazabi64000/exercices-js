Solution Exercice 3

Code fonctionnel

setTimeout(function() {
  let message = "Bonjour après 2 secondes";
  console.log(message);
}, 2000); // 2000 millisecondes = 2 secondes

Cours simplifié

- setTimeout : permet d’exécuter une fonction après un certain temps (en millisecondes).
- Ici, la fonction s’exécute après 2000 millisecondes (2 secondes).
- console.log() : affiche le message dans la console.

Exemple :

setTimeout(function() {
  console.log("3 secondes sont passées !");
}, 3000); // Affiche après 3 secondes
