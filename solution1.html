Solution Exercice 1

Code fonctionnel

let result = "Bonjour !";
console.log(result); // Affiche "Bonjour !" dans la console

Cours simplifié

- let : sert à créer une variable pour stocker une valeur.
- console.log() : affiche le contenu dans la console.
- Ici, on crée une variable result qui contient "Bonjour !" et on l'affiche.

Exemple :

let nom = "Alice";
console.log("Salut " + nom + " !");

Cela affichera : Salut Alice !
