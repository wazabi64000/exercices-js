Solution Exercice 9

function somme(tab) {
  return tab.reduce((acc, n) => acc + n, 0);
}
console.log(somme([1,2,3])); // Affiche 6

Cours simplifié :

  • tab.reduce((acc, n) => acc + n, 0) : additionne tous les éléments du tableau.
  • acc : accumulateur (somme actuelle).
  • n : élément actuel du tableau.
