Solution Exercice 2

Code fonctionnel

function direBonjour() {
  return "Salut, bienvenue !";
}
console.log(direBonjour()); // Affiche "Salut, bienvenue !"

Cours simplifié

- function : sert à créer une fonction qui peut exécuter du code plusieurs fois.
- On peut donner un nom à la fonction, ici direBonjour.
- return : permet de renvoyer une valeur quand on appelle la fonction.
- console.log() : affiche le résultat dans la console.

Exemple :

function addition(a, b) {
  return a + b;
}
console.log(addition(3, 4)); // Affiche 7
