Solution Exercice 5

Code fonctionnel

let texte = localStorage.getItem("texte");
console.log(texte);

Cours simplifié

- localStorage.getItem("clé") : récupère une donnée stockée.
- Si rien n’est enregistré, on peut afficher un message par défaut.
- console.log() et document.getElementById(...).textContent permettent de voir le résultat.

Exemple :

let nom = localStorage.getItem("nom");
console.log(nom || "Aucun nom trouvé");
