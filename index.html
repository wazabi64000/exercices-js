Mini-Projet JavaScript pour Débutants

Bienvenue ! Ce mini-projet contient 10 exercices progressifs pour apprendre les bases de JavaScript : fonctions, setTimeout, setInterval, stockage local et manipulation du DOM.

Chaque exercice possède une page dédiée et une solution avec un mini-cours simplifié.

Pour commencer, cliquez sur un exercice dans la navbar.
