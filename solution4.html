Solution Exercice 4

Code fonctionnel

let texte = document.getElementById('monTexte').value;
localStorage.setItem("texte", texte);
console.log("Texte enregistré !");

Cours simplifié

- localStorage.setItem("clé", valeur) : stocke une donnée dans le navigateur.
- document.getElementById('monTexte').value : récupère le texte écrit par l’utilisateur.
- console.log() : affiche un message dans la console.

Exemple :

localStorage.setItem("nom", "Alice");
console.log(localStorage.getItem("nom")); // Affiche "Alice"
