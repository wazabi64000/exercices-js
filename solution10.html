Solution Exercice 10

let count = 0;
setInterval(() => {
  count++;
  console.log(count);
}, 1000);

Cours simplifié :

  • setInterval(func, ms) : exécute func toutes les ms millisecondes.
  • count++ : incrémente le compteur.
  • document.getElementById('compteur').textContent = count : affiche le compteur dans la page.
  • clearInterval(interval) : permet d’arrêter un compteur si nécessaire.
